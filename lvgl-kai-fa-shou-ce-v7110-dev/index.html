简介

LVGL (light and Versatile Graphics Library) 是免费开源的图形库，提供了开发嵌入式 GUI 所需的一切，具有简单易用的图形元素、绚丽的视觉效果、精简的内存占用等特点。

特点

  • 开源免费 MIT
  • 支持 PC 模拟器
  • 控件丰富，高度自定义
  • 高级图形特效支持
  • 多种输入设备支持、多语言支持（UTF-8）、多显示器支持
  • C 语言开发，兼容 C++，支持 MicroPython
  • 存储要求（>64 kB Flash，>16 kB RAM）
  • 可使用操作系统、外扩存储、GPU，不依赖具体硬件（MCU、显示器）

要求

要求 推荐
处理器 16、32、64 bit
时钟 >16 MHz
ROM >64 kB >180 kB
RAM/static ~2 kB
RAM/stack >2 kB >8 kB
RAM/heap >2 kB >16 kB

存储占用可能依赖于架构、编译器、编译选项。
